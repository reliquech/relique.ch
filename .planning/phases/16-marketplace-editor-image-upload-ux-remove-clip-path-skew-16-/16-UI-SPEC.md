## Visual Hierarchy & Focal Point

- **Focal Point:** The 16:9 Cover Image container is the primary visual anchor on the screen.
- **Visual Hierarchy:** Cover Image preview/dropzone -> Additional Images grid -> Upload state indicators.
- **Accessibility Fallbacks:** All icon-only buttons (e.g., Close `X` button, Delete trash button, navigation arrows) must have explicit `aria-label` attribute descriptions.
