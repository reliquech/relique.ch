  <action>
Install devDependency: `@next/bundle-analyzer`.

Wrap `next.config.js` export:
```js
import bundleAnalyzer from '@next/bundle-analyzer';
const withBundleAnalyzer = bundleAnalyzer({ enabled: process.env.ANALYZE === 'true' });
export default withBundleAnalyzer(nextConfig);
```

Add scripts to `package.json`:
```json
"analyze": "cross-env ANALYZE=true npm run build",
"phase11:bundle-report": "node scripts/phase11-bundle-report.mjs",
"phase11:bundle-baseline": "node scripts/phase11-bundle-report.mjs --write-baseline"
```

Create `scripts/phase11-bundle-report.mjs`:
- Walk `.next/static/chunks/*.js` (skip if `.next` missing — print error)
- Sum total KB, list top 15 largest chunks with filenames
- `--write-baseline` writes `.planning/phases/11-javascript-bundle-performance-baseline-audit-tree-shakeable-/11-baseline-chunks.json`
- Default mode prints table to stdout

Use `cross-env` only if already in project; otherwise use `set ANALYZE=true&&` pattern documented in 11-BASELINE.md for Windows.
