Build the Marketplace editor media workflow with drag/drop uploads, progress, retry/remove, reorder, alt text, and primary image selection.
</objective>

<threat_model>

T13-14: Large files can produce unclear failure states. Surface per-file errors and retry actions without blocking the rest of the editor.
</threat_model>
