  <action>
    POST `/api/public/consign` — no requireUser:

    1. Parse `request.formData()`
    2. Honeypot: reject 400 if `website` field non-empty (spam guard per RESEARCH)
    3. Validate text fields with Zod: contact_name, contact_email, contact_phone?, item_description (min 10), category?, coa_issuer?
    4. Photos: `formData.getAll("photos")` as File[] — min 1, max 10, each max 10MB, mime jpeg/png/webp only
    5. Insert `consigned_items` status=`submitted` via service role
    6. Upload photos to `consign-submissions/{submissionId}/{uuid}.{ext}` via service role storage.upload
    7. Insert `attachments` rows: entity_type=`consigned_item`, entity_id=submission id, file_path=storage path
    8. Insert `leads`: full_name=contact_name, email=contact_email, phone=contact_phone, source=`consign`, status=`new`
    9. Send confirmation email to contact_email via sendTransactionalEmail (English transactional copy)
    10. Return 201 `{ id, status: "submitted" }`

    Set `export const maxDuration = 60` for large uploads.

    Do NOT use `/api/consigned` — keep admin route auth-required (D-09).
  </action>
